# Placeholders

## Purpose

Temporary stand-in art used during development or testing before a real
Product Asset is ready — e.g. a generic "coming soon" illustration used
anywhere a real Product Illustration hasn't shipped yet. Distinct from a
World's own placeholder preview images (like Museum Gallery's early
plain-swatch `preview.png`), which live inside that World's own package,
never here.


## Ownership

VihuStudio product/design (temporary, by definition).


## Expected Asset Types

- placeholder-coming-soon.webp
- placeholder-illustration.webp

## Naming Convention

`placeholder-<what-its-standing-in-for>.webp`. A placeholder is removed the moment its real asset ships — it should never become a permanent dependency.


## Example Usage

Screen 1's "More Ideas" card using `placeholder-coming-soon.webp` until a real illustration for that tile is produced.
