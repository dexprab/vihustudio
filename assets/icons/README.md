# Icons

## Purpose

Functional UI icons — small glyphs that communicate an action or state,
as opposed to `illustrations/`, which are narrative/emotional and much
larger. Split by role: navigation/, actions/, status/.

Note: `logo.png` already exists directly inside this folder from before
this sprint's convention was established. It predates the Product Asset
System and is left in place rather than moved; the brand mark's
permanent home going forward is `assets/brand/`, not here.
