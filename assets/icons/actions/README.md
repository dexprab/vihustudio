# Actions

## Purpose

Action icons — add, import, delete, edit.


## Ownership

VihuStudio product/design.


## Expected Asset Types

- icon-action-add.svg
- icon-action-import.svg
- icon-action-delete.svg

## Naming Convention

`icon-action-<name>.svg`.


## Example Usage

The "⊕ Add New World" button (`.creation-flow-add-world-header-btn`) currently renders a plain `⊕` character — this is its illustrated replacement.
