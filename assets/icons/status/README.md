# Status

## Purpose

Status/feedback icons — selected checkmark, success, warning.


## Ownership

VihuStudio product/design.


## Expected Asset Types

- icon-status-selected.svg
- icon-status-success.svg
- icon-status-warning.svg

## Naming Convention

`icon-status-<name>.svg`.


## Example Usage

The gold checkmark badge on a selected World/Page-Style card (`.creation-flow-card-check`, currently a plain `✓` character) is this icon's current placeholder.
