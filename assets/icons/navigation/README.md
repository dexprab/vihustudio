# Navigation

## Purpose

Navigation icons — back arrows, chevrons, close buttons.


## Ownership

VihuStudio product/design.


## Expected Asset Types

- icon-nav-back.svg
- icon-nav-chevron-right.svg
- icon-nav-close.svg

## Naming Convention

`icon-nav-<name>.svg`.


## Example Usage

Screen 2's "← Back" button and its horizontal-scroll "›" chevron (`.creation-flow-back`, `.creation-flow-scroll-btn` in `css/style.css`) currently render plain text glyphs — these icons are their illustrated replacement.
