# Stickers

## Purpose

Sticker artwork for Sticker Studio's global catalog (Sprint 8.x/9.6 —
categories like Social and Shape). Stickers are a Studio-wide catalog a
child can add to any card, so they are a Product Asset even though a
specific sticker's *placement* on a page is per-project data — the
artwork itself is never owned by one World.


## Ownership

VihuStudio product/design.


## Expected Asset Types

- sticker-shape-star.webp
- sticker-social-heart.webp

## Naming Convention

`sticker-<category>-<name>.webp`, category matching `js/stickerLibrary.js`'s existing category ids.


## Example Usage

A new sticker category's artwork added here, then registered in `js/stickerLibrary.js`'s catalog (a Studio Runtime change — out of this sprint's scope, listed here only so the asset's home is unambiguous when that sprint happens).
