# Illustrations

## Purpose

Home of the permanent **Product Illustration** family documented in
`docs/PRODUCT_ILLUSTRATIONS.md` — the six Creation Type illustrations
(Tell a Story, Showcase My Artwork, Create Quotes, Write a Poem, Greeting
Cards, More Coming Soon) and any future illustration in that same family.
Split into ready-to-use exports (`creation/`) and editable originals
(`source/`).
