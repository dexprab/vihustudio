# Source

## Purpose

The original, editable source files each `creation/` export is generated
from (layered files, not flattened exports). Never referenced by the
running app directly — only by whoever next edits an illustration.


## Ownership

VihuStudio design team.


## Expected Asset Types

- creation-story.psd / .procreate / .svg (layered)
- one source file per creation/ export, same base name

## Naming Convention

Same base name as its exported counterpart in `creation/`, with the source tool's own extension: `creation-story.procreate`.


## Example Usage

Opening `creation-story.procreate` to touch up the Tell a Story illustration, then re-exporting over `creation/creation-story.webp`.
