# Product Assets

## Purpose

The root of VihuStudio's **Product Asset System** — every visual asset owned
by the product itself, as opposed to assets owned by an individual World
(see `docs/PRODUCT_ASSETS.md` for the full Product vs. World distinction).
Nothing in this tree travels inside a `.vtheme` package; nothing a World
owns is duplicated here.


## Ownership

VihuStudio (the product). Not any single World, theme, or sprint.


## Expected Asset Types

- Nothing directly — this folder only contains the category subfolders below.

## Naming Convention

Subfolder names are fixed by `docs/PRODUCT_ASSETS.md`; do not add new top-level categories without updating that document first.


## Example Usage

See each subfolder's own README for what it holds and how it's referenced.
