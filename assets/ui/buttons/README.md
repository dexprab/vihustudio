# Buttons

## Purpose

Hand-painted button background art or textures, for surfaces where a plain CSS pill (the current approach — see `.creation-flow-start-btn` etc.) is replaced with illustrated button art.


## Ownership

VihuStudio product/design.


## Expected Asset Types

- button-primary.webp
- button-primary-pressed.webp

## Naming Convention

`button-<variant>.webp`, with a `-pressed` / `-hover` suffix for interaction states.


## Example Usage

Start Creating's pill button gaining a textured watercolor background instead of a flat CSS fill.
