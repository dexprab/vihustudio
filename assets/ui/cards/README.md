# Cards

## Purpose

Illustrated card background textures, for surfaces where a plain CSS card (Creation Type cards, World cards, Page Style cards) gains a hand-painted paper background instead of a flat fill.


## Ownership

VihuStudio product/design.


## Expected Asset Types

- card-default.webp
- card-selected.webp

## Naming Convention

`card-<state>.webp`.


## Example Usage

A Creation Type card's background swapped from `var(--cf-paper)` to a textured paper image.
