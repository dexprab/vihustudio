# Shadows

## Purpose

Soft, hand-drawn shadow art for lifting cards and panels, for surfaces where a plain CSS `box-shadow` (the current approach throughout) is replaced with an illustrated soft-shadow asset.


## Ownership

VihuStudio product/design.


## Expected Asset Types

- shadow-card.webp
- shadow-panel.webp

## Naming Convention

`shadow-<what-it-lifts>.webp`.


## Example Usage

A World card's hover-lift shadow drawn from art instead of `box-shadow:0 6px 18px rgba(0,0,0,0.08)`.
