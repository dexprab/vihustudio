# Frames

## Purpose

Decorative UI frame art belonging to Studio chrome itself — e.g. a frame
around the Selected World Preview's hero artwork. Not to be confused with
a World's own Frame Variations (Picture Frame styles like Classic White
Mat / Gold Accent), which are a Theme concept and travel with the World,
never stored here.


## Ownership

VihuStudio product/design.


## Expected Asset Types

- frame-preview-hero.webp

## Naming Convention

`frame-<where-it's-used>.webp`.


## Example Usage

An ornate border drawn around `.creation-flow-preview-hero` instead of its current plain `border-bottom`.
