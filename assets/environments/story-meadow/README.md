# Story Meadow

## Purpose

Assets specific to the Story Meadow environment: sky, rolling hills,
grass, trees, the wooden signpost, the distant castle. Story Meadow is
currently implemented as pure CSS/emoji (`.creation-scene` in
`index.html` + its CSS in `css/style.css`) — this folder is where a
hand-painted replacement for that placeholder would live, asset by
asset, once produced.


## Ownership

VihuStudio product/design.


## Expected Asset Types

- story-meadow-sky.webp
- story-meadow-hills-back.webp / -front.webp
- story-meadow-tree-01.webp
- story-meadow-signpost.webp
- story-meadow-castle.webp

## Naming Convention

`story-meadow-<element>.webp`, numbered when there are repeatable variants: `story-meadow-tree-01.webp`, `story-meadow-tree-02.webp`.


## Example Usage

Replacing `.creation-scene-sky`'s CSS gradient with `background-image:url('assets/environments/story-meadow/story-meadow-sky.webp')`.
