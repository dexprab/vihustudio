# Butterflies

## Purpose

Butterfly sprites shared across every environment, for the same reason as clouds/ — one shared set, not one per environment.


## Ownership

VihuStudio product/design.


## Expected Asset Types

- butterfly-01.webp
- butterfly-02.webp
- butterfly-orange-01.webp

## Naming Convention

`butterfly-<number>.webp`, or `butterfly-<colour>-<number>.webp` when colour variants exist.


## Example Usage

`.creation-scene-butterfly` (currently the 🦋 emoji with a CSS flutter animation) would use one of these sprites, keeping the same flutter keyframes.
