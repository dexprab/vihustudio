# Flowers

## Purpose

Flower sprites for ground-level meadow decoration.


## Ownership

VihuStudio product/design.


## Expected Asset Types

- flower-daisy.webp
- flower-tulip.webp
- flower-sunflower.webp

## Naming Convention

`flower-<common-name>.webp`.


## Example Usage

`.creation-scene-flower` (currently 🌼/🌸/🌷/🌻 emoji with a CSS sway animation) would use one of these sprites per instance.
