# Clouds

## Purpose

Cloud sprites shared across every environment — both Story Meadow and Open Sky drift clouds through their sky layer, and both should draw from the same set rather than each maintaining their own.


## Ownership

VihuStudio product/design.


## Expected Asset Types

- cloud-01.webp
- cloud-02.webp
- cloud-03.webp

## Naming Convention

`cloud-<number>.webp`, two-digit, in rough size order (smallest first).


## Example Usage

`.creation-scene-cloud`'s CSS shapes (currently drawn with layered `border-radius` circles) would be replaced by one of these sprites.
