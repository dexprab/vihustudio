# Open Sky

## Purpose

Assets for the alternative Open Sky environment named alongside Story
Meadow in the canonical storyboard and documented in
`docs/ENVIRONMENTS.md`. Not yet built in the running app — Story Meadow
is the one live implementation today.


## Ownership

VihuStudio product/design.


## Expected Asset Types

- open-sky-sky.webp
- open-sky-clouds-backdrop.webp
- open-sky-horizon.webp

## Naming Convention

`open-sky-<element>.webp`, same numbering convention as story-meadow/.


## Example Usage

A future `.creation-scene--open-sky` variant would source its layers from this folder the same way Story Meadow does from its own.
